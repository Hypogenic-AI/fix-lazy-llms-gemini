# Cloned Repositories

## 1. ProphetNet (CRITIC)
- **URL**: https://github.com/microsoft/ProphetNet
- **Location**: code/ProphetNet
- **Relevant Directory**: `code/ProphetNet/CRITIC`
- **Purpose**: Implementation of the "CRITIC" framework where LLMs self-correct using external tools.
- **Key Relevance**: Provides a baseline and methodology for "Self-Correction" which is related to the "Harsh Critic" hypothesis (critiquing own outputs).

## 2. TALE (Token-Budget-Aware LLM Reasoning)
- **URL**: https://github.com/GeniusHTX/TALE
- **Location**: code/TALE
- **Purpose**: Implementation of token-budget-aware reasoning.
- **Key Relevance**: Directly addresses the "Response Budget" aspect of the research hypothesis. It controls the verbosity/laziness by enforcing budgets.
